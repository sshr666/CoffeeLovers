# Coffee Lovers Team Portfolio

A static coffee-themed portfolio website for the Coffee Lovers onboarding assignment.

## Local Preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8028 --bind 127.0.0.1
```

Then visit:

```text
http://127.0.0.1:8028/
```

## GitLab Pages

This folder includes `.gitlab-ci.yml`, so it can be hosted on GitLab Pages by pushing this folder as the root of a GitLab repository. GitLab will publish the static site from the generated `public` artifact.
